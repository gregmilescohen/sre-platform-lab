# Pulseboard API

## Overview
Backend API + Database interaction service for the Pulseboard platform.

## Tech Stack
- Python (FastAPI)
- PostgreSQL
- Docker

## Testing
- Run tests with: pytest
- Coverage: Run pytest --cov

## Deployment
- Deployed via Docker
- CI/CD via GitHub Actions (see .github/workflows/)

## For AI Agents
- This is a standard Python FastAPI backend service
- Follow existing patterns in the codebase for adding endpoints
- Ensure proper error handling and logging
- Write tests for new functionality
